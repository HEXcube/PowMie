# PwoMie
Squish your goals with PwoMie, the cutest Pomodoro timer!

[Open PwoMie!](https://HEXcube.github.io/PwoMie)

## Local Development

Prerequisites:
- [Node.js](https://nodejs.org/) (Version 24+)

```bash
# Install dependencies
npm install

# Start local dev server
npm run dev

# Run production build
npm run build

# Preview production build locally
npm run preview
```

## Deployment
This project is configured to be deployed automatically to GitHub Pages via GitHub Actions upon every push to the `main` branch.
