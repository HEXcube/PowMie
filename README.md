# PowMie
Squish your goals with PowMie, the cutest Pomodoro timer!

[Open PowMie!](https://HEXcube.github.io/PowMie/)

## Local Development

Prerequisites:
- [Node.js](https://nodejs.org/) (Version 18+)

```bash
# Install dependencies
npm install

# Start local dev server
npm run dev

# Run production build
npm run build

# Preview production build locally
npm run preview
```

## Deployment
This project is configured to be deployed automatically to GitHub Pages via GitHub Actions upon every push to the `main` branch.
